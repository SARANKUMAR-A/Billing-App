# 💳 Smart Billing System

A full-stack billing application built using **Django (Backend)** and **React.js (Frontend)** with PostgreSQL support.

---

## 🚀 Features

- 🧾 Generate bills with tax calculation
- 📦 Product selection with search dropdown
- ➕ Add / Remove products dynamically
- 💰 Denomination-based cash handling
- ⚠️ Validation for payment & denominations
- 📊 Auto total calculation
- 📤 Upload products via Excel
- 📧 Email invoice to customer
- 🔄 Real-time UI updates

---

## 🏗️ Tech Stack

### Backend
- Django
- Django REST Framework
- PostgreSQL
- Pandas (Excel upload)

### Frontend
- React.js
- Material UI (MUI)
- Axios
